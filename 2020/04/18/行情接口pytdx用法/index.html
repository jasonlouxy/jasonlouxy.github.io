api 方法

  • 获取股票行情：需要传入一个列表，每个列表由一个市场代码， 一个股票代码构成的元祖构成 [ (市场代码1， 股票代码1)，(市场代码2， 股票代码2) … (市场代码n， 股票代码n) ]，如：

注意点：非股票品种代码，有些获取的价格不是实际价格，比如可转债获取价格为实际价格 x 10。这是可能是TDX为了防止浮点数错误，报价在传输和存储时实际都保存为整数，然后根据品种进行处理的结果。

获得的结果是个OrderedDict，字段包含：

  • 获取k线：api.get_security_bars(type, market, code, start, count)
    type：0，5分钟；1，15分钟；2，30分钟；3，小时；4，日线，5，周线；6，月线；10，季线；11，年线
    market：0，深圳；1，上海
    start：指定的范围开始位置
    count：要求的k线数目，最大值为800
  • 获取市场股票数量：api.get_security_count(market)
  • 获取股票列表：api.get_security_list(market, start)
  • 获取指数k线：api.get_index_bars(type, market, code, start, count)
  • 查询分时行情：api.get_minute_time_data(market, code)
  • 查询历史分时行情：api.get_history_minute_time_data(market, code, date)

注：在引入TDXParams之后，from pytdx.params import TDXParams，可以使用 TDXParams.MARKET_SH 和 TDXParams.MARKET_SZ 作为常量来替代1和0作为参数

  • 查询分笔成交：api.get_transaction_data(market, code, start, count)
  • 查询历史分笔成交：api.get_history_transaction_data(market, code, start, count, date)
  • 查询公司信息目录：api.get_company_info_categroy(market, code)
  • 读取公司信息详情：api.get_company_info_content(market, code, filename, start, count)
  • 读取除权除息信息：api.get_xdxr_info(market, code)，结果是个list，列出每年的分红、送股、配股、缩股、行权
  • 读取财务信息：api.get_finance_info(market, code)
